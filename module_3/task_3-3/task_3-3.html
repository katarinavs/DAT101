1. 1. What is the difference between invoking and creating a function?

    2. Creating a function means defining it, while invoking a function means executing the code inside the function.

  2. 2. Explain the syntax regarding semicolons and code blocks.

    3. Semicolons are used to conclude statements. Code blocks are used to group statements together.

  3. 3. What happens when we try to store values from a function when it doesn't return any?

    4. The variable you use to store the function's result will hold the value 'undefined'.

  4. 4. How do we separate parameters from each other when creating a function?

    5. You separate parameters from each other bby using commas within the parentheses of the functions definition.

  5. 5. Explain how parameter names are related during the invocation and creation of a function.

    6. During creation you define parameters as placeholders for the values within the function definition. During invocation you pass arguments in the same order as the parameters are defined.

  6. 6. Explain why a parameter is undefined when a function call is missing a parameter.

    7. Function parameters does not have default values unless you explicity set them. Therefore when you call a function without providing the expected arguments, the parameter will be 'undefined'.

  7. 7. Explain what happens when we use recursive functions.

    8. Recursive functions are functions that call themselves in order to solve a problem.
